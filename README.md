# eth-brute

## Installing

```
git clone git@github.com:rust3r/eth-brute.git
cd eth-blute/
go get
go build
```

## Running
For help
```
./eth-brute -h

Usage of ./eth-brute:
  -pk string
    	Start private key
  -threads int
    	Number of threads (default 4)

```
Example
```
./eth-brute -threads 50 -pk 1206b75e20883f695a49bbabd1f26e5d30afb75e4a9e3989a89d779d4a3a1c92
```
